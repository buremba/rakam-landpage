When you install Rakam API, you will get a feature-rich RESTFul API that you can analyze your data. You can use client libraries or simply execute JSON requests to Rakam API in order to run SQL queries, execute behavioural analysis queries such as funnel and retention easily. Better yet, we have another product Rakam BI which is highly-integrated with Rakam API. You can use it for creating custom reports, dashboards and performing admin operations.
