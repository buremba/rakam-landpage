Rakam API

Rakam is a modular analytics platform that provides you a set of features to create your own analytics service without hassle. Rakam API is free and open source so that you can just start to deploy right away to your cloud provider, on-premise servers or just local environment.

  • Rich analytical features with more than 100 endpoints.
  • Designed to handle your events concurrently at any scale.
  • One-click installation to AWS, Heroku, Digitalocean or Docker.
  • It's modular, enable the features that you want to use or develop custom modules for your needs.


Rakam UI

You can build custom reports or custom dashboards just connect Rakam with third-party tools or join Rakam data with internal data sources. A full stack analytics platform for you, including both backend and frontend.

  • Customize for your needs, it's flexible.
