rakam Api

Rakam is a modular analytics platform that provides you a set of features to create your own analytics service without hassle.

One-click analytics backend

Rakam API is an open-source analytics backend with more then 100 endpoints. It exposes a RESTFul API with more than 100 endpoints for both collecting and analyzing data.

We use Swagger for defining API endpoints so Rakam API can be integrated into your current infrastructure. You can also develop custom modules and implement new endpoints and features depending on your needs.

You can directly connect your Rakam API via RESTFul API or Rakam API client library for your favorite programming language. Our SDKs are also built on top of Rakam API so you can easily set your Rakam API and send your data directly to your Rakam API Cluster.

Rich analytical features with more than 100 endpoints

One-click installation to AWS, Heroku, Digitalocean or Docker

Designed to handle your events concurrently at any scale. Both horizontally and vertically scalable.

It's modular, enable the features that you want to use or develop custom modules for your needs.

Fast and scalable

A single Rakam API server can handle more than 50K events concurrently, it's based on Java 8, Netty and Jackson. We use our custom JSON parsers in order to use lower memory footprint and CPU, Netty allows us to handle thousands of requests concurrently without any problem. You can also scale your Rakam API by adding new nodes to the cluster, we provide extra tools for scaling your Rakam API cluster using your favorite cloud provider.
