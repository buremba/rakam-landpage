Advanced BI for
Rakam API

Rakam UI is the frontend of your Rakam API, it connects your Rakam API and visualize the data for you. It allows you to run event explorer, funnel, retention queries on your event data-set collected via Rakam API and also allows you to create custom reports, dashboards with advanced SQL editor and simple query builder interface.

You can create your custom analytics service without writing any single code with Rakam UI. The developers and analytsts can create custom reports and dashboards and share them with the teams in your company.

We take security seriously. Rakam UI doesn't actually have accesss to your Rakam APIs, your browser executes the AJAX requests directly on your Rakam API. Rakam UI can only store metadata of your reports, not your data.

Security is not a feature,
it's state of mind.

Rakam UI is not just a website, it works as an application. Your Rakam API may be behind a firewall or even your localhost, You can use Rakam UI if your computer is able to connect your Rakam API but it't not accessible from outside because Rakam UI uses your browser to connect Rakam API via AJAX requests. It just stores metadata of your reports, not data.
